# 원시자료형(Primitive Type)

- 정수,실수,문자,문자열을 표현할 때 사용하는 8가지 **기본자료형**

| 이름    | 분류            | 사이즈(byte 수) |
| ------- | --------------- | --------------- |
| byte    | 정수            | 1byte           |
| short   | 정수            | 2byte           |
| int     | 정수            | 4byte           |
| long    | 정수 (접미어L)  | 8byte           |
| float   | 실수 (접미어F)  | 4byte           |
| double  | 실수 (접미어 D) | 8byte           |
| char    | 단일문자        | 2byte           |
| boolean | 논리(참과 거짓) | 1byte           |

1 byte = 8 bit

정수 기본형 int / 실수 기본형 double
